### React: Referenzen auf nativen DOM
    - `React.createRef() ` erzeugt ein allgemeines Referenz-Objekt - Das Objekt wird in der Komponente gespeichert (i.d.R. als Instanz-Variable) - Eine Komponente wird der Referenz zugewiesen - Über `current`-Property am Referenz-Objekt erhält man Zugriff auf das native DOM-Element 

  class HelloMessage extends React.Component {
    constructor(props) { 
      super(props);
      this.inputRef = React.createRef();
    }
    render() {
      return 

          <input ref={this.inputRef} 
                    onChange={...} value={...}/>
          <button onClick={() => this.reset()}>Clear</button>
        
;
    }
  
    reset() {
      this.setState({greeting: ""});
  
      // focus() ist Funktion auf nativem DOM Element
      this.inputRef.current.focus();
    }
  }
  `
